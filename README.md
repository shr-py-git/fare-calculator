Tourists visiting India often face travel fare scams due to a lack of awareness about actual distances between locations. Local transport providers may exploit this information gap by charging inflated fares. This problem is especially prevalent in major cities where tourists rely on taxis, autos, or informal transport services. There is currently no simple, reliable, and easily accessible tool that allows tourists to independently verify the real travel fares between two places. This lack of transparency results in financial exploitation and a loss of trust in local transport systems.


---

ðŸ’¡ SOLUTION OVERVIEW

FairRoute is a lightweight web-based prototype that enables users to input a source and destination and instantly view the actual distance between them. By providing accurate distance data using open mapping services, FairRoute empowers tourists to make informed decisions and verify whether a quoted fare is reasonable. The solution promotes transparency and fairness in local travel pricing.
